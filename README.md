## How to Run `bot.py`

### 1. Setup
- Download and extract the project as a ZIP file
- Open the extracted folder

### 2. Create Emoji Folder
- In the same directory as `bot.py`, create a new folder
- Name it whatever is set in:
  EMOJI_FOLDER = "YOUR_NAME"

### 3. Configure the Bot
Open `bot.py` and set the following values:

- Bot Token:
  TOKEN = "BOT_TOKEN"

- Guild (Server) ID:
  GUILD_ID = YOUR_GUILD_ID

### 4. Add Bot to Your Server
- Invite the bot using the Discord Developer Portal

### 5. Add Emojis
- Place all required emoji files into the emoji folder you created

### 6. Run the Bot
python bot.py
